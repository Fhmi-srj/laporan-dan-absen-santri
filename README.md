# Laporan dan Absen Santri

Sistem manajemen laporan aktivitas dan absensi santri berbasis PHP.

## Fitur Utama
- Dashboard statistik
- Manajemen data siswa
- Pencatatan aktivitas (Sakit, Izin Keluar, Izin Pulang, Sambangan, Pelanggaran, Paket, Hafalan)
- Sistem absensi dengan QR Code dan RFID
- Integrasi WhatsApp untuk notifikasi ke wali
- Laporan dan export data

## Teknologi
- PHP 7.4+
- MySQL/MariaDB
- Bootstrap 5
- jQuery
- DataTables

## Instalasi
1. Clone repository
2. Import database dari folder `sql/`
3. Konfigurasi `config.php`
4. Akses via browser

## Konfigurasi
Edit file `config.php` untuk mengatur:
- Database connection
- WhatsApp API credentials
- Upload settings
